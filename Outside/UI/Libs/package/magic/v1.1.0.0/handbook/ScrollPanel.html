滚动面板

具备纵向滚动条的面板，多用于页面模块化中的一个特定大小的区域使用，滚动条可以根据需求自定义样式。

参考手册

参数配置

  • options.autoUpdateDelay : Number
    自动适应内容区域大小延时（ms）， 默认值 500。如果您的内容区域并非动态，设置为 false 可减少一个定时器开销
  • options.arrowButtonStep : Number
    点击箭头按钮滚动的距离（px）， 默认值 20
  • options.mousewheelStep : Number
    鼠标滚轮滚动的距离（px）， 默认值 50
  • options.scrollbarStep : Number
    点击滚动条空白区域滚动的距离（px）， 默认值 80
  • options.intervalScrollDelay : Number
    （点击到箭头或滚动条空白区域时）持续滚动延时（ms），默认 300
  • options.intervalScrollFreq : Number
    （点击到箭头或滚动条空白区域时）持续滚动频率（ms），默认 100
  • options.scrollbarMinHeight : Number
    滚动条控制手柄最小高度（px），默认 10

组件属性

组件方法

  • update

    根据变化了的内容区域，重新计算滚动条(是否出现、位置、高度等)

  • clearAutoUpdate

    停止自动更新

  • scrollTo

    滚动到某个位置

    • 参数

      • pos: {Number}

        要滚动到的位置

  • scrollToTop

    滚动到顶部

  • scrollToBottom

    滚动到底部

  • scrollToElement

    滚动到某个元素的位置

    • 参数

      • ele: {Element}

        要滚动到的元素

  • scrollBy

    以当前位置为基准滚动

    • 参数

      • num: {Number}

        要滚动的距离

  • getScroll

    获取当前滚动的位置

  • getScrollPct

    获取当前滚动的百分比位置

  • getElement

    取得 ui 模块对应的 dom element 对象

    • 参数

      • key: {String}

        该ID对应的关键字(可选参数)

    • 返回值: {Object}

      ui模块对应的dom element

  • getElements

    取得所有 ui 模块对应的 dom element 对象

    • 返回值: {Object}

      取得对象的集合

组件事件

  • beforescroll

    滚动前触发

    • 参数

      • e: {event}

      • e.pos: {Number}

        要滚动到的位置

  • afterscroll

    滚动后触发

    • 参数

      • e: {event}

      • e.pos: {Number}

        滚动后的位置
