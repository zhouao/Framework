气泡提示

飘浮在页面主体内容上的层，多用于提醒和提示。气泡提示具有一个箭头，可以设置箭头样式和位置。

气泡提示插件

参考手册

参数配置

  • target : Array
    需要提示的节点。(必选)

组件属性

组件方法

  • render

    渲染提示框

    • 参数

      • el: {HTMLElement|id|dom}

        渲染到目标容器下，如果该值存在，则提示框组件在el下，如果不存在且目标节点存在的情况下，提示框组件和目标节点是兄弟关系，如果都不存在，则渲染到body下。

  • reposition

    重新定位提示框位置

  • show

    显示提示框

  • hide

    隐藏提示框

  • setTarget

    设置目标节点

  • setContent

    设置提示框内容

  • setPosition

    设置提示框位置

组件事件

  • onload

    提示框渲染完时触发

  • onbeforeshow

    显示提示框之前触发,如果返回为false,则阻止显示。

  • onshow

    显示提示框时触发

  • onbeforehide

    隐藏提示框之前触发,如果为false,则阻止隐藏。

  • onhide

    隐藏提示框时触发

  • ontargetchange

    当目标节点发生变化时触发。
